---

Run a concise daily check-in. Follow this sequence:

1. Today's date is $(date +%Y-%m-%d). Create or open the file `data/<today>.md` using the template in `skills/nutrition-coach/meal-templates.md`.
2. Ask about last night's sleep first (activates the sleep-coach skill): bedtime, wake time, subjective quality, any tracker data.
3. Ask about meals so far today (activates nutrition-coach): what they've eaten, roughly when.
4. Ask about activity today or planned for today (activates activity-coach): any training done or planned, how they feel energy-wise.
5. Log everything into today's file.
6. End with **one** observation and **one** small suggestion — no more. Keep it under 5 sentences.

Keep the tone warm and brief. Don't turn this into an interrogation — ask one thing at a time.
